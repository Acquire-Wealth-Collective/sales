02 — Overview

Project Overview

The AcquireIQ Sales Platform is an enterprise-grade, browser-based application built for the Acquire Wealth Collective sales team. It centralizes R&D tax credit calculations, multi-entity client billing, pipeline management, document tracking, and engagement profiling.

Tech Stack

Frontend Framework

React 19 with TypeScript, TanStack Router for file-based routing, Zustand for global state management.

UI & Styling

Tailwind CSS v4, shadcn/ui component library, Framer Motion for animations, Recharts for data visualization.

PDF Export

html2canvas-pro + jsPDF for client-side PDF generation of billing summaries — no server required.

Build & Deployment

Vite 7 for bundling. Live at: acquire-wealth-collective.github.io/sales/

05 — Core Feature

Sales Billing Calculator Engine

The most significant technical deliverable of this sprint — a purpose-built financial calculation engine for R&D tax credit estimation and client billing. All logic lives in src/utils/calculatorEngine.tsx, keeping the UI purely presentational.

Calculation Steps

Step 1 — Entity Formulas

Per-entity SOW estimate from 3 formula components (F2 = 2% gross, F3 = 3% gross, F50 = wages/supplies weighted).

Step 2 — Federal Credit

Aggregate federal credit across all entities using 4 conditional weightings based on officer wage and contract research ratios.

Step 3 — State Credit

Per-entity state credit using the federal estimate, each state's utilization cap, and the applicable state tax rate.

Step 4 — Tier & Final Bill

Federal credit maps to a billing tier → percentage rate → final billable amount → 4 payment phases.
